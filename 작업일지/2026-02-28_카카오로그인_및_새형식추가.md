## 📝 특이사항 및 해결 과정

### 카카오 SDK 버전 이슈
**문제**: 최신 버전(2.7.2)에서 `window.Kakao.Auth.login` 함수 미작동
**해결**: 구 버전 SDK로 변경 → 성공

### 리다이렉트 vs 팝업 방식
**초기 시도**: `Kakao.Auth.authorize()` 리다이렉트 방식
**문제**: 돌아온 후 인증 코드 처리 로직 필요
**최종 해결**: `Kakao.Auth.login()` 팝업 방식 사용 → 간단하고 효과적

### 로그인 상태 관리
**요구사항**: Firebase Auth와 카카오 로그인 통합
**해결**: 
- localStorage에 카카오 사용자 정보 저장
- AuthContext에서 통합 관리
- 페이지 새로고침 시에도 상태 유지
